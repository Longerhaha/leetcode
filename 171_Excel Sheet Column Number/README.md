## 171_Excel Sheet Column Number（Excel表列序号）
## 问题描述与输入输出
给定一个Excel表格中的列名称，返回其相应的列序号。

	A -> 1
	B -> 2
	C -> 3
	...
	Z -> 26
	AA -> 27
	AB -> 28 
	...

### 问题示例

	示例1：
	输入：
	"A"
	输出：
	1

	示例2：
	输入：
	"AB"
	输出：
	28
	
	示例3：
	输入：
	"ZY"
	输出：
	701


函数输入与输出：
* 输入：
	* char* s: Excel表格中的列名称的一级指针。

* 输出：
	* int : 相应的列序号

## 思路			
### 常规法
	
	相当于算26进制数，较为简单。
