## 118_Pascal's Triangle（杨辉三角）
## 问题描述与输入输出
给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。
在杨辉三角中，每个数是它左上方和右上方的数的和。

### 问题示例

	示例1：
	输入：
	5
	输出：
	[
		 [1],
		[1,1],
	   [1,2,1],
	  [1,3,3,1],
	 [1,4,6,4,1]
	]
	

函数输入与输出：
* 输入：
	* int numRows：给定的行数目
	* int** columnSizes: *columnSizes是指向每行的个数（数组）的一级指针，故columnSizes是二级指针。
	
* 输出：
	* int** : 生成的杨辉三角的数据的二级指针。

## 思路			
### 对称性法

	从上到下一层一层击破。
	1.先描述左右两个1；
	2.计算左半部分的值；
	3.根据对称性补足右半部分的值。
